The Franckate Cipher API turns any text into a structured ciphertext you can decode by hand. Perfect for learning cryptography — one API call shows you exactly how it works.

plaintext Hello
→

quickstart.py 
import requests

API_KEY = "your_api_key_here"
BASE    = "https://franckate-api.onrender.com/api"

# Encrypt
res = requests.post(
    f"{BASE}/encrypt",
    headers={"X-API-Key": API_KEY},
    json={"text": "Hello World"}
)

data = res.json()
print(data["encrypted"])
# → U7.L4.L11.L11.L14.F27.U22.L14.L17.L11.L3.

How the cipher works

Franckate is a substitution cipher. Every character maps to a category prefix + its position in that category. Dot-separated. Human-readable. Teachable.

U
Uppercase
A → U0. · B → U1. · Z → U25.
L
Lowercase
a → L0. · e → L4. · z → L25.
D
Digits
0 → D0. · 5 → D5. · 9 → D9.
F
Symbols & Space
! → F0. · space → F27. · ? → F22.
Decode this by hand
U7.L4.L11.L11.L14.F27.F22.
Show answer
U7 = H · L4 = e · L11 = l · L11 = l · L14 = o · F27 = (space) · F22 = ?
→ "Hello ?"

API Reference

All endpoints accept JSON bodies and return JSON. Authenticate with your API key in the X-API-Key header.

Base URL https://franckate-api.onrender.com
POST /api/encrypt 🔑 API Key

Encrypt any plaintext string using the Franckate cipher.

{
  "text": "Hello World"
}
FieldTypeRequiredDescription
textstringYesPlaintext to encrypt (max 10,000 chars)
{
  "original":  "Hello World",
  "encrypted": "U7.L4.L11.L11.L14.F27.U22.L14.L17.L11.L3.",
  "length": {
    "input":  11,
    "output": 42
  },
  "algorithm": "franckate-v1"
}
curl -X POST https://franckate-api.onrender.com/api/encrypt \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{"text": "Hello World"}'
import requests

response = requests.post(
    "https://franckate-api.onrender.com/api/encrypt",

POST /api/decrypt 🔑 API Key

Decrypt Franckate ciphertext back to plaintext.

{
  "text": "U7.L4.L11.L11.L14."
}
{
  "original":  "U7.L4.L11.L11.L14.",
  "decrypted": "Hello",
  "algorithm": "franckate-v1"
}
curl -X POST https://franckate-api.onrender.com/api/decrypt \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{"text": "U7.L4.L11.L11.L14."}'
POST /api/encrypt/steps 🔑 API Key Educational

Returns a character-by-character breakdown of the encryption process. Perfect for building cryptography tutorials.

{
  "original":  "Hi",
  "encrypted": "U7.L8.",
  "total_steps": 2,
  "steps": [
    {
      "position":            0,
      "character":           "H",
      "category":            "uppercase",
      "category_code":       "U",
      "index_in_category":   7,
      "token":               "U7.",
      "explanation":  "'H' is uppercase at index 7 → 'U7.'"
    },
    ...
  ]
}
POST /api/batch/encrypt 🔑 API Key

Encrypt up to 50 strings in a single request.

{
  "texts": ["Hello", "World", "Crypto"]
}
{
  "count": 3,
  "algorithm": "franckate-v1",
  "results": [
    { "index": 0, "original": "Hello", "encrypted": "U7.L4.L11.L11.L14." },
    { "index": 1, "original": "World", "encrypted": "U22.L14.L17.L11.L3." },
    { "index": 2, "original": "Crypto", "encrypted": "U2.L17.L24.L15.L19.L14." }
  ]
}
POST /api/analyze 🔑 API Key

Analyze the character distribution of a text before encrypting — great for understanding cipher expansion ratios.

{
  "analysis": {
    "length": 11,
    "unique_characters": 8,
    "counts": { "uppercase": 2, "lowercase": 8, "digits": 0, "special": 1 },
    "percentages": { "uppercase": 18.2, "lowercase": 72.7, ... }
  }
}
GET /api/info Public

Returns API metadata — no key required. Good for discovery and health checks.
